# Content Audit Framework - JustxEmpower

## Audit Methodology

For each page, we will:
1. **Content Editor Check** - Document all sections and fields in the CMS
2. **Live Site Check** - Document all visible content on the live page
3. **Code Check** - Verify if content is from CMS (usePageContent) or hardcoded
4. **Triangulation** - Compare all three sources to find discrepancies

## Audit Categories

### Issue Types
- **ORPHAN** - Section exists in CMS but not rendered on live site
- **HARDCODED** - Content on live site that should come from CMS but is hardcoded
- **REDUNDANT** - Duplicate sections serving the same purpose
- **MISSING** - Content visible on live site but no CMS entry exists
- **MISMATCH** - CMS field exists but wrong content key used in code
